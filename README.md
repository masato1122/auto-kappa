Auto-ALAMODE2
=====================

This is a software for an automated calculation of thermal conductivity with VASP and ALAMODE.

Prerequisites
----------------

* VASP
* Alamode (alm>=1.4.1, anphon>=1.4.1, and ALM>=1.4.0)
* Phonopy   >= 2.15.1
* ASE       >= 3.22.1
* Pymatgen  >= 2022.3.24
* Spglib    >= 1.16.5
* Custodian >= 2022.5.26
