How to use Git
=================

To make branch
---------------

> git branch
> git branch develop
> git checkout develop
> git push origin develop


To make feature
----------------

> git branch
> git branch feature/aaa
> git checkout feature/aaa

* If develop branch was modified
> git rebase develop

* When the feature was modified
> git add .
> git commit -m "message"
> git push origin feature/aaa

* To merge the feature to the develop branch
> git checkout develop
> git merge feature/aaa
> git branch -d feature/aaa

To merge to main
------------------
> git checkout main
> git merge develop
> git push origin main
