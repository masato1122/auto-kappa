2. How to use auto-kappa

2.1. Examples

  1. Example for auto-kappa

The .../examples/5_single directory contains an example to compute thermal conductivity of Silicon. While the run.sh script can be used to run this calculation, it is recommended to use a job scheduler to run this example. An example job script is shown below. Please modify it according to your environment.

#!/bin/sh
#PBS -q default
#PBS -l nodes=1:ppn=24  ## Only single node calculation is available
#PBS -j oe
#PBS -N test            ## job name

export LANG=C
export OMP_NUM_THREADS=1  ## Please set OMP_NUM_THREADS=1
cd $PBS_O_WORKDIR

nprocess=24               ## Number of processes

mpid=mp-149
dir_db=${directory_of_downloaded_phoonondb}/${mpid}  ## This line must be modified.

if [ ! -e $dir_db ]; then
    echo " Cannot find $dir_db"
    exit
fi

akrun \
    --directory $dir_db \
    --outdir $mpid \
    --nprocess $nprocess
