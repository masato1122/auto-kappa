2. Examples

2.1. 1. Examples for required packages

Example jobs are available in the auto-kappa/examples directory for alm (examples/1_alm), anphon (examples/2_anphon), ASE (examples/3_ase), and Custodian (examples/4_custodian). Please go to each directory and run the jobs.

2.2. 2. auto-kappa example

The .../examples/5_single directory contains an example to compute thermal conductivity of Silicon using auto-kappa. An example job script is shown below. Since all processes are included in this job, it takes about half an hour to complete. It is recommended to use a job scheduler to submit this job. Please modify it according to your environment.

Listing 2.2.1 read this file as an example job script to run examples/5_single
 1### Description
 2# volume_relaxation : relaxation calc. with the equation of state
 3# max_natoms        : maximum number of atoms in a supercell
 4# nmax_suggest      : maximum number of displacement patterns for the FD method
 5
 6### Abbreviation
 7# FC : force constant
 8# FD : finite displacement
 9
10nprocs=2
11
12### Input for each material
13file_structure=./POSCAR.Si
14mpid=mp-149   ## material ID for Silicon
15
16### Commands
17c_mpirun=mpirun
18c_vasp=vasp
19c_alm=alm
20c_anphon=anphon
21
22### Variables
23vol_relax=1     ## Structural relaxation with the equation of state
24nmax_suggest=1  ## Maximum number of displacement patterns for the FD method. If 1, always use LASSO approach.
25max_natoms=150  ## Maximum number of atoms in a supercell
26
27### Check command existence
28for command in $c_mpirun $c_vasp $c_alm $c_anphon; do
29    if ! command -v $command &> /dev/null; then
30        echo "$command could not be found"
31        exit
32    fi
33done
34
35### Run auto-kappa
36akrun \
37    --file_structure $file_structure \
38    --outdir $mpid \
39    --nprocs $nprocs \
40    --mpirun         $c_mpirun \
41    --command_vasp   $c_vasp \
42    --command_alm    $c_alm \
43    --command_anphon $c_anphon \
44    --volume_relaxation $vol_relax \
45    --max_natoms   $max_natoms \
46    --nmax_suggest $nmax_suggest \
47    --calculate_forces 1
